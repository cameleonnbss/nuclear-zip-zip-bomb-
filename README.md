# 💣 ZipBomb

> **Entertainment purposes only** — handle with care.

A zip bomb built for fun and learning, inspired by the legendary [42.zip](https://www.unforgettable.dk/).

---

## What is a zip bomb?

A zip bomb is a compressed archive that looks tiny but expands to a massive size when decompressed — sometimes petabytes from just a few kilobytes. It works by using recursive or deeply nested compression layers, exhausting the system's memory and disk space.
